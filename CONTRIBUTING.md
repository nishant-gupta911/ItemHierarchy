# Contributing to Item Processing System

Thank you for your interest in contributing to the Item Processing System project!

## Getting Started

1. Clone the repository
2. Follow the setup instructions in README.md
3. Create a new branch for your feature: `git checkout -b feature/FeatureName`
4. Make your changes and commit with clear messages
5. Push to your branch and submit a pull request

## Code Standards

- Follow C# naming conventions (PascalCase for classes/methods, camelCase for fields/parameters)
- Write XML documentation for public classes and methods
- Add appropriate error handling and logging
- Use parameterized queries for database operations
- Write meaningful commit messages

## Testing

- Test all CRUD operations before submitting
- Verify circular dependency detection works correctly
- Test with both valid and invalid inputs
- Check database connectivity in development environment

## Database Changes

- Update setup.sql with any schema changes
- Document migration steps in commit messages
- Test with fresh database setup

## Reporting Issues

Please provide:
- Clear description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Environment details (OS, .NET version, SQL Server version)

## Code Review Process

- Maintainers will review all pull requests
- Changes may be requested before merging
- Once approved, your contribution will be merged

Thank you for contributing!
